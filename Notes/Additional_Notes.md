# **Additional Questions and Answers**

---

## **1. What is a training dataset?**

A training dataset is a collection of labeled examples used to teach a machine learning model how to identify patterns. It forms the primary data from which the model learns its parameters during training.

---

## **2. What is test data?**

Test data is an unseen dataset used to evaluate how well a trained model performs on new inputs. It helps measure the modelâ€™s generalization ability and real-world effectiveness.

---

## **3. What is validation test?**

A validation test uses a separate dataset to fine-tune model hyperparameters and prevent overfitting. It acts as an intermediate evaluation step between training and final testing.

---

## **4. Explain Decision Tree Classifier.**

A Decision Tree Classifier is a predictive model that assigns class labels by splitting the data into branches based on attribute tests. It follows a tree-like structure where each leaf represents a final class prediction.

---

## **5. Explain the architecture of Artificial Neural Network.**

The architecture of an ANN consists of input, hidden, and output layers connected through weighted links. Each neuron applies an activation function and passes signals forward to enable pattern learning and representation.

---

## **6. Explain the training and testing algorithm for Backpropagation Neural Network.**

### **Training Algorithm**

* Initialize all network weights with small random values to prepare for learning.
* Perform feedforward computation to obtain outputs from input samples.
* Compute error by comparing predicted output with actual target values.
* Apply backpropagation to calculate gradients and update weights using learning rate.

### **Testing Algorithm**

* Provide unseen test samples to the trained network as inputs.
* Perform feedforward propagation to generate predicted outputs.
* Compare predictions with true labels to measure accuracy and generalization.

---

## **7. Explain the different activation functions.**

Activation functions transform input signals in neurons and introduce nonlinearity to the network. Popular functions include sigmoid for smooth binary outputs, ReLU for efficient deep learning, tanh for centered activations, and softmax for multi-class probability distribution.

---

## **8. What are the different multivariate methods for learning? Explain.**

Multivariate learning methods analyze multiple input variables simultaneously to understand complex relationships. Techniques include multivariate regression for predicting continuous values, multivariate classification for labeling data, logistic regression for binary decisions, and discriminant analysis for class separation.

---

## **9. What is regression?**

Regression is a supervised learning method used to predict continuous numerical outputs based on input variables. It models the relationship between features and target values to estimate future trends or measurements.

---

## **10. Explain convolutional neural network (CNN).**

A Convolutional Neural Network is a deep learning model that uses convolutional layers to extract spatial features from images. It captures edges, textures, and complex patterns, making it powerful for image classification and recognition tasks.

---

## **11. What is LSTM?**

Long Short-Term Memory is a specialized recurrent network designed to store information over long sequences using memory cells and gates. It effectively handles long-term dependencies in tasks like language processing and time-series prediction.
